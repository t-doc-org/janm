# Objectifs de l'examen

- Bases du langage JavaScript :
  - Inclure du code JavaScript dans une page HTML à l'aide de la balise `<script>`
- Variables :
  - Déclarer une variable avec le mot-clef `let`
  - Modifier la valeur d'une variable existante sans répéter `let`
  - Afficher une valeur ou une variable avec `console.log()`
- Branchements conditionnels :
  - Écrire un `if` avec la condition entre **parenthèses** et le bloc entre **accolades**
  - Utiliser `else` et `else if`
  - Combiner des conditions avec les opérateurs logiques `&&` (et) et `||` (ou)
  - Lire une saisie utilisateur avec `prompt(...)` et la convertir en nombre avec `Number(...)`
- Fonctions :
  - Définir une fonction avec le mot-clef `function`
  - Appeler une fonction par son nom suivi de `()`
- Manipulation du HTML (DOM) :
  - Donner un identifiant unique à une balise avec l'attribut `id`
  - Récupérer une balise en JavaScript avec `document.getElementById("...")`
  - Lire et modifier le texte d'une balise avec la propriété `textContent`
  - Lire et modifier l'apparence d'une balise avec la propriété `style`
- Événements :
  - Placer un bouton dans une page avec la balise `<button>`
  - Lier une action à un événement avec `addEventListener("click", maFonction)`
